◆ callback_t

callback_t

A helper typedef to make working with callbacks easier.

An example callback:

void callback(struct mg_connection* connection,
struct http_message* message,
void* user_data) {
printf("callback triggered!\n");
}
Parameters
connectionThe Mongoose server's HTTP connection with a client.
messageThe Mongoose HTTP message sent from the client that triggered the callback.
user_dataA void pointer to the user data pased when the server was created.
